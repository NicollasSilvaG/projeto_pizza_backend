TYPEORM SETUP
1) As seguintes dependencias devem ser instaladas
typeorm, reflect-metadata

DependÃªncias dev(-D): typescript, ts-node-dev ou ts-node


2) Salve o seguinte script para poder realizar os comandos em seguida:	

    "typeorm": "ts-node-dev ./node_modules/typeorm/cli.js"
	
	ou
	
	"typeorm": "ts-node ./node_modules/typeorm/cli.js"

3) Comandos

Criar Migration:
npm run typeorm migration:create src/db/typeorm/migrations/CreateNomeTabela

Rodar Migration: 
npm run typeorm -- -d ./src/db/typeorm/index.ts migration:run 

Reverter Migration
npm run typeorm -- -d ./src/db/typeorm/index.ts migration:revert
